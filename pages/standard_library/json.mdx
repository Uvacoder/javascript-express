# JSON

JSON, or **J**ava**S**cript **O**bject **N**otation, is a subset of JavaScript that only includes: `Boolean`, `Number`, `String`, `Array`, `Object`, `null`

There are standard library methods for serializing from and to strings: `JSON.parse` and `JSON.stringify`
